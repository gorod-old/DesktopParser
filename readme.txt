Использование:
Распакуйте архив и запустите parser.exe.
Отключить звук кнопка SOUND ON/OFF (сигнал запуска - оканчания парсинга).
Кнопка AUTO EXCEL в положении ON автоматически запускается Excel при сохранке (сохранка - кнопка SAVE).
Сохранять можно в любой момент не зависимо от того закончен парсинг или нет. Результаты в папке "result data", 
названия файлов по времени и дате.
Файлы user-agents.txt и proxies.txt необходимы для установки списка прокси адресов и юзер агентов соответственно -
находятся в папке text_files. Юзер агенты по умолчанию установлены. 
В файлы user-agents.txt и proxies.txt можно добавлять свои значения (каждое значение с новой строки).
Пример:
178.32.116.64:3128
178.124.208.8:3128
51.222.21.93:32768
189.112.111.194:80
2.179.193.146:80
Если используете прокси с авторизацией формат должен быть следующий:
login:password@ip:port
Для примера: ukrci:jrdopiu@178.32.116.64:3128
Парсинг с прокси активируется кнопкой PROXY ON/OFF.
При отсутствии списка прокси в proxies.txt активация невозможна.
Для корректной работы нужен установленный Microsoft Office (у меня стоит 2021 Pro но скорее всего разницы нет какой, не проверял)

В случае зависания (не закрывается окно парсера, приложение не отвечает...) Диспетчер задач -> parser -> снять задачу
